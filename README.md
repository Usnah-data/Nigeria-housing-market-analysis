#  Nigeria Housing Market Analysis Dashboard

##  Project Overview
This project analyzes housing market trends across Nigeria using Power BI. The dashboard provides insights into pricing, affordability, geographic inequality, and property distribution.

## Problem Statement
Housing affordability and price distribution vary significantly across Nigerian states. This project aims to identify:
- Price disparities across regions
- Affordability challenges
- Market structure and property distribution
- Luxury vs standard housing trends


##  Key Metrics
- **Average Price:** ₦301.38M  
- **Median Price:** ₦85.00M  
- **Affordability Index:** 28.33  
- **Price Per Bedroom:** ₦69.46M  


##  Key Insights
-  Significant **geographic inequality** in housing prices across states
-  High **average price vs median price gap** → indicates presence of luxury outliers
-  Nearly **45% of listings are luxury properties**
-  **Detached duplexes dominate** the market listings


##  Dashboard Features
- Market overview KPIs
- Median prices by state
- Luxury vs standard housing distribution
- Property type breakdown
- Interactive filters (state, title, luxury category)


##  Tools Used
- Power BI
- Excel / CSV dataset
- Data Cleaning & Transformation
- Data Visualization
